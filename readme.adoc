# SICK App Designer Docs

This repository provides documentation and helpful resources for the SICK App Designer.

SICK App Designer is an extension pack published on the Visual Studio Code Marketplace. It provides all extensions that are required to develop apps for programmable devices of the SICK ecosystem.

NOTE: Documentation for the predecessor SICK AppSpace SDK can be found https://github.com/SICKAG/SICK-AppSpace-SDK-Docs/tree/archive_sick-appspace-sdk[here]

## How to give feedback

If you encounter a bug or an issue with the SICK App Designer development tools for VSCode please https://jira.sick.com/rest/collectors/1.0/template/form/5e947ee6[Report a bug] at our Jira project.
If you're missing features feel free to  https://jira.sick.com/rest/collectors/1.0/template/form/08bd7384[Create a feature request] at our Jira project.
